Installation
==============

You can install telegram.py from PyPI.

.. note::
    Python 3.10 or higher is required to use the library.

.. tab:: Unix (Mac/Linux)

    .. code-block:: shell

        python3 -m pip install telegram.py

.. tab:: Windows

    .. code-block:: shell

        py -3 -m pip install telegram.py
